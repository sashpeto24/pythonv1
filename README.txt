ПРОЕКТ: АВТОСАЛОН НА PYTHON

Описание:
Това е конзолна Python програма за управление на автосалон.

Функционалности:
- Добавяне на автомобил
- Преглед на всички автомобили
- Търсене по марка
- Сортиране по цена
- Продажба на автомобил
- Промяна на цена

Файлове:
- models.py -> съдържа класа Car
- main.py -> основна логика и меню
- requirements.txt -> използвани библиотеки
- README.txt -> описание на проекта

Стартиране:
1. Инсталирайте Python.
2. Отворете терминал в папката на проекта.
3. Стартирайте командата:

python main.py
